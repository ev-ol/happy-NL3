## 📖 Sobre 

Este é um projeto desenvolvido durante a **[Next Level Week](https://nextlevelweek.com/)**, realizada pela **[@Rocketseat](https://github.com/Rocketseat)** durante os dias 12 a 18 de Outubro de 2020.

A proposta do projeto é uma aplicação fictícia que incentive qualquer um a visitar o orfanato mais próximo e ter uma experiência incrível fazendo o dia de uma ou mais crianças melhor. O objetivo, além do aprendizado com código, é demonstrar através dessa experiência, que há muito na vida para ser grato todos os dias.
